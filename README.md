# sovereign-ai-tclass-2.0
T-Class 2.0: The Standard Framework for AI Sovereignty and Technical Independence
> **소버린 AI 주권 등급 표준 2.0**

## 📖 개요 (Introduction)
최근 AI 모델의 유사성 논란과 기술 유출 공방을 보며, 우리는 소모적인 논쟁을 넘어 **"무엇이 진정한 기술 주권(Sovereignty)인가?"**에 대한 명확한 기준을 정립해야 합니다.

이 프로젝트는 글로벌 트렌드와 기술적 실체(Engineering Reality)를 반영하여, AI 모델의 주권 수준을 판별하는 **[7단계 등급 체계 (T-Class 2.0)]**를 정의합니다.

이 기준은 단순한 성능 평가가 아니라, **기술의 원천, 통제권, 그리고 안보적 가치**를 기준으로 하며, 향후 정부 및 기업의 AI 전략 수립을 위한 나침반이 되는 것을 목표로 합니다.
